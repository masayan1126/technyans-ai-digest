# X Post Generator Skill

記事のパスを指定して、テクにゃん.のXアカウント用のポスト投稿案を生成するスキル。

## 使い方

```
/x-post src/content/articles/YYYYMMDD/article-slug/ja.md
```

## 出力

- 日本語版 3パターン
- 英語版 3パターン
- 記事URL、ハッシュタグ、OGP画像添付指示を含む
