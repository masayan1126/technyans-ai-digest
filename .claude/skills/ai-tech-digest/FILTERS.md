# フィルタリングルール

## 関連性スコアリング

検索結果を以下の基準で評価:

### High Priority (必ず含める)
- 優先テーマの製品アップデート
- 新機能リリース
- Breaking changes
- 重要なAPI変更
- メジャーバージョンアップ

### Medium Priority (内容次第)
- チュートリアル・使い方記事
- 活用事例
- パフォーマンス改善
- バグフィックス

### Low Priority (スキップ可)
- 一般論・概説記事
- 既知情報の再掲
- マイナーな不具合報告

## 除外パターン

以下を含む記事は除外:

```regex
# セールス・マーケティング
- "今すぐ購入"
- "限定オファー"
- "割引コード"

# 低品質コンテンツ
- "○○の衝撃的な真実"
- "誰も教えてくれない"
- clickbait系タイトル

# 無関係なトピック
- AI倫理のみの議論 (実装を伴わない)
- 暗号資産 (AI関連でない)
- 株価予測
```

## ドメインフィルタ

### 信頼できるドメイン (優先)
```
anthropic.com
openai.com
deepmind.google
github.com
cursor.sh
techcrunch.com
venturebeat.com
theverge.com
arstechnica.com
```

### 除外ドメイン
```
# スパム・低品質サイト
- コンテンツファーム
- アフィリエイトまとめサイト
- 自動生成コンテンツサイト
```

## コンテンツ品質チェック

最終的に以下を確認:

1. **実用性**: 具体的な使い方や実装が含まれているか
2. **新規性**: 過去1週間以内の情報か、新しい知見か
3. **信頼性**: 出典が明確か、公式情報か
4. **関連性**: 定義したテーマに直接関連しているか

## アクションフラグ

記事分類:

- `MUST_INCLUDE`: 最優先で記事化
- `RECOMMENDED`: 通常記事として含める
- `OPTIONAL`: スペースがあれば含める
- `SKIP`: 除外
