# Microsoft Azure: The Only Cloud Offering Both Claude and GPT

On November 18, 2025, Microsoft announced a groundbreaking strategic partnership with NVIDIA and Anthropic, making Azure the **only cloud platform in the world** that provides customers access to both Anthropic's Claude and OpenAI's GPT frontier models on a single platform.

## Historic Partnership Details

### Investment Commitments
The partnership involves unprecedented investment amounts:
- **NVIDIA**: Up to $10 billion investment in Anthropic
- **Microsoft**: Up to $5 billion investment in Anthropic
- **Anthropic**: $30 billion commitment to purchase Azure compute capacity
- **Additional capacity**: Plans to contract compute "up to one gigawatt"

### What Makes Azure Unique

Azure is now the **only cloud platform** where enterprises can access both:
1. **Claude models** (Anthropic): Sonnet 4.5, Opus 4.1, and Haiku 4.5
2. **GPT models** (OpenAI): GPT-4, GPT-4o, and future releases

This unique positioning eliminates the need for enterprises to maintain multi-cloud infrastructure just to access different frontier AI models.

## Claude Models Now Available in Microsoft Foundry

Three Claude versions are available in **public preview** through serverless deployment:

### Claude Sonnet 4.5
- Positioned as "the best coding model in the world"
- Sophisticated reasoning and multi-step workflows
- Autonomous coding capabilities
- Ideal for complex development tasks

### Claude Haiku 4.5
- Fastest model in the Claude family
- Near-frontier capabilities at ~1/3 of Sonnet's cost
- Perfect for high-volume applications
- Use cases: Customer support automation, data processing

### Claude Opus 4.1
- Specialized for complex reasoning tasks
- Sustained focus on multi-step problems
- Enterprise-grade performance
- Best for critical analysis and decision-making

## Enterprise Benefits

### 1. Unified Procurement
- **Microsoft Azure Consumption Commitment (MACC)**: Claude usage counts toward existing Azure commitments
- **No separate contracts**: Work directly with current Azure agreements
- **Simplified billing**: One invoice for all AI services
- **Faster procurement**: No additional vendor approval processes

### 2. Integrated Ecosystem
- **Microsoft Entra authentication**: Seamless security integration
- **Multiple SDKs**: Python, TypeScript, and C# support
- **Existing workflows**: Integrate with current Azure infrastructure
- **Copilot family access**: Claude available in GitHub Copilot, Microsoft 365 Copilot, and Copilot Studio

### 3. Advanced Capabilities
Claude models in Foundry support:
- **Code execution**: Run and test code within the model
- **Web search**: Real-time information retrieval
- **Vision capabilities**: Image understanding and analysis
- **Tool use**: Integration with external APIs and services
- **Prompt caching**: Improved performance and cost efficiency

## Deployment Options

### Global Standard Deployment
- Currently rolling out to customers
- Standard API pricing model

### US DataZone
- Coming soon for data sovereignty requirements
- Compliant with US data residency regulations

## Technical Collaboration

### NVIDIA Partnership
NVIDIA and Anthropic will collaborate on:
- **Hardware optimization**: Design and engineering for Anthropic models
- **Performance tuning**: Optimize for efficiency and total cost of ownership
- **Future architectures**: Co-design next-generation NVIDIA systems for AI workloads
- **Target systems**: Grace Blackwell and Vera Rubin compute platforms

### Microsoft Integration
Microsoft has committed to:
- **Continuing Claude access** across all Copilot products
- **Deep platform integration** with Azure infrastructure
- **Enterprise support** and SLAs for production workloads

## Multi-Cloud Strategy

While Azure is the only platform offering both Claude and GPT together, this partnership makes Anthropic the **first major AI provider available on all three major cloud platforms**:
- **Microsoft Azure**: Claude + GPT (unique combination)
- **Amazon Web Services (AWS)**: Claude via Amazon Bedrock
- **Google Cloud Platform (GCP)**: Claude via Vertex AI
