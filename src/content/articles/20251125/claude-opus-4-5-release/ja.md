Anthropicは2025年11月24日、最新のフラッグシップモデル「Claude Opus 4.5」を正式リリースしました。これは、9月のSonnet 4.5、10月のHaiku 4.5に続く、4.5シリーズの最後のモデルとなります。

## コーディング性能で王座を奪還

Opus 4.5は、「コーディング、エージェント、Computer Useで世界最高のモデル」として位置付けられており、OpenAIのGPT-5.1-Codex-MaxやGoogleのGemini 3といった最新の競合モデルに対抗する形でリリースされました。

主要なベンチマーク結果:
- **SWE-bench Verified**: 77.2%(標準設定)、82.0%(高計算設定)
- **OSWorld**: 61.4%(実際のコンピュータ使用タスク)
- トークン使用量を半減させながら、内部コーディングベンチマークで優れた性能を達成

## 技術仕様

- **コンテキスト長**: 200,000トークン(Sonnet 4.5と同等)
- **出力制限**: 64,000トークン
- **知識カットオフ**: 2025年3月(Sonnet 4.5は1月、Haiku 4.5は2月)
- **価格**: 入力$5/100万トークン、出力$25/100万トークン(前世代Opusクラスモデルの1/3)

## 主要プラットフォームでの展開

Opus 4.5は、リリースと同時に複数の主要プラットフォームで利用可能になりました:

### Microsoft エコシステム
- **Microsoft Foundry**: East US2、Sweden Centralリージョンでパブリックプレビュー開始
- **GitHub Copilot**: Pro、Pro+、Business、Enterpriseユーザー向けに提供(2025年12月5日まで1xプレミアムリクエスト倍率)
  - VS Code(Agent、Plan、Ask、Edit mode)
  - github.com
  - GitHub Mobile(iOS/Android)
  - Copilot CLI
- **Microsoft Copilot Studio**: 統合対応

### その他のプラットフォーム
- **Databricks**: DBSQLとAgent Bricksで利用可能
- **Anthropic自社サービス**: Pro、Max、Enterpriseプランでデフォルトモデルとして採用

## 新機能とプロダクト統合

Opus 4.5のリリースと合わせて、Anthropicは以下の新機能を発表しました:

### Claude for Chrome
パイロット版から全Maxユーザーへ一般提供を開始。ブラウザ内でのAI支援機能を提供します。

### Claude for Excel
Max、Team、Enterpriseユーザー向けに提供。スプレッドシート操作に特化した機能を実装しています。

### Claude Code のアップグレード
- **改良されたプランモード**: より正確な問題解決計画を作成し、計画により忠実に実行
- **デスクトップアプリ対応**: Claude Codeがデスクトップアプリでもサポート

## AI安全性への取り組み

Anthropicのシステムカードによると、Opus 4.5は「これまでにリリースした中で最も堅牢にアライメントされたモデル」であり、あらゆる開発者による最高クラスのフロンティアモデルであるとしています。

評価では、人間の悪用への協力と、モデルが独自に行う望ましくない行動の両方を含む、広範な「問題行動」スコアで測定され、より安全で安全なモデルへの傾向を継続しています。

## 業界への影響

Opus 4.5のリリースは、わずか数週間前のOpus 4.1リリースに続くもので、生成AIにおける急速なイノベーションペースを示しています。

プロフェッショナルなソフトウェア開発者、金融アナリスト、コンサルタント、会計士などの知識労働者が理想的なユーザーとして想定されており、エンタープライズ向けの複雑なタスクに対応する能力が強調されています。

## まとめ

Claude Opus 4.5は、性能、価格、実用性のバランスを大幅に改善したモデルとして登場しました。特に:

- **性能**: コーディングとエージェント機能で世界最高クラス
- **コスト**: 前世代の1/3の価格で提供
- **実用性**: Chrome、Excel統合により日常業務への組み込みが容易
- **エコシステム**: 主要プラットフォームで即座に利用可能

AI競争が激化する中、Anthropicは価格と性能の両面で競争力を示し、エンタープライズ市場への本格的な参入を加速させています。
