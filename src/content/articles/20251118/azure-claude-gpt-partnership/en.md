# Microsoft Azure Becomes the Only Cloud Offering Both Claude and GPT

## Historic Announcement

On November 18, 2025, Microsoft announced a groundbreaking achievement: **Azure is now the only cloud platform in the world** that provides customers access to both Anthropic's Claude and OpenAI's GPT frontier models on a single platform.

This announcement came alongside a massive $45 billion partnership involving Microsoft, NVIDIA, and Anthropic, marking one of the largest strategic alliances in the AI industry.

## The $45 Billion Partnership Details

### Investment Breakdown

The partnership involves unprecedented financial commitments:

- **NVIDIA**: Up to $10 billion investment in Anthropic
- **Microsoft**: Up to $5 billion investment in Anthropic
- **Anthropic**: $30 billion commitment to purchase Azure compute capacity
- **Extended commitment**: Plans to contract additional compute capacity "up to one gigawatt"

### What Makes This Unique

This is the first time any cloud provider can offer customers both leading frontier model families - Claude and GPT - on a single platform with unified infrastructure, billing, and security.

**Previously:**
- Enterprises needed separate contracts with AWS (for Claude) + Azure (for GPT)
- Or Azure (for GPT) + Google Cloud (for Claude)
- Managing multiple cloud vendors, billing systems, and security configurations

**Now with Azure:**
- Single platform for both Claude and GPT
- One contract, one billing system
- Unified security and compliance
- Simplified procurement through Microsoft Azure Consumption Commitment (MACC)

## Claude Models Available on Azure

Three Claude versions are now available in **public preview** through Microsoft Foundry with serverless deployment:

### Claude Sonnet 4.5
**"The world's best coding model"**

- Advanced reasoning and multi-step workflows
- Autonomous coding capabilities
- Sophisticated problem-solving
- Ideal for: Complex development tasks, code generation, technical analysis

### Claude Haiku 4.5
**Fastest and most cost-efficient**

- Near-frontier capabilities at ~1/3 of Sonnet's cost
- Optimized for high-volume applications
- Rapid response times
- Ideal for: Customer support automation, data processing, real-time applications

### Claude Opus 4.1
**For complex reasoning tasks**

- Specialized for sustained multi-step reasoning
- Enterprise-grade performance
- Deep analytical capabilities
- Ideal for: Critical analysis, strategic planning, complex decision-making

## Advanced Features Available

All Claude models on Azure Foundry support:

- **Code execution**: Run and test code within the model environment
- **Web search**: Real-time information retrieval and fact-checking
- **Vision capabilities**: Image understanding, analysis, and description
- **Tool use**: Integration with external APIs and services
- **Prompt caching**: Improved performance and reduced costs for repeated queries

## Enterprise Benefits

### 1. Unified Procurement & Billing

**Microsoft Azure Consumption Commitment (MACC) Integration**
- Claude usage counts toward existing Azure commitments
- No separate vendor contracts needed
- Single invoice for all AI services (Claude + GPT + other Azure services)
- Faster procurement - no additional approval processes

This alone can save enterprises months of procurement time and significant administrative overhead.
